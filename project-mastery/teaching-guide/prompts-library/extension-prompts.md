# 功能扩展提示词库 🚀

用于添加新功能的提示词模板。

---

## 数据库扩展

### 1. 添加新数据模型

```
我想在项目中添加新的数据模型。

模型名称：[名称]
用途：[描述用途]

需要的字段：
- [字段1]: [类型] - [说明]
- [字段2]: [类型] - [说明]

与现有模型的关系：
- 与 [模型] 是 [一对多/多对多/一对一] 关系

请提供：
1. Prisma schema 定义
2. 迁移命令
3. 基本的 CRUD Server Actions
```

### 2. 添加新字段

```
我想给 [模型] 添加新字段。

新字段：
- 名称: [字段名]
- 类型: [String/Int/Boolean/DateTime/Decimal]
- 必填: [是/否]
- 默认值: [值]

请提供：
1. Schema 修改
2. 迁移步骤
3. 相关代码更新建议
```

## 功能模块扩展

### 3. 添加数据导出功能

```
参考项目现有的交易系统，帮我添加数据导出功能。

需求：
- 导出格式: [Excel/CSV/PDF]
- 导出内容: [描述要导出的数据]
- 触发方式: [按钮点击]

请提供完整实现，包括：
1. Server Action
2. 前端按钮和处理逻辑
3. 所需的 npm 包
```

### 4. 添加搜索功能

```
我想添加全局搜索功能。

搜索范围：
- [表1]: [字段1, 字段2]
- [表2]: [字段1, 字段2]

UI位置：[头部/侧边栏]

参考项目的代码风格，请提供完整实现。
```

## UI/UX 扩展

### 5. 添加暗色模式

```
我想为项目添加暗色/亮色主题切换。

项目使用 Tailwind CSS。

请提供：
1. 主题切换实现
2. 本地存储持久化
3. Tailwind 配置更新
4. 示例组件改造
```

### 6. 添加数据可视化图表

```
我想添加[类型]图表展示[数据]。

数据来源：[描述]
图表类型：[柱状图/折线图/饼图]
位置：[页面路径]

项目已安装 Recharts。
请提供完整的图表组件代码。
```
