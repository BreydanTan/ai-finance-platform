# 调试提示词库 🐛

常见错误的调试提示词模板。

---

## 常见错误类型

### 1. Module not found

```
我遇到 "Cannot find module" 错误：

错误信息：
[粘贴完整错误]

我的 package.json 中有这个包：[是/否]
我运行过 npm install：[是/否]

请告诉我如何解决。
```

### 2. Prisma 错误

```
Prisma 操作失败：

错误：
[粘贴错误]

我的 schema.prisma：
[粘贴相关部分]

最近的操作：[描述最近做了什么]

请帮我诊断问题。
```

### 3. 认证错误

```
Clerk 认证不工作：

症状：[描述问题]
环境变量：[已配置/未配置]
错误信息：[粘贴错误]

请提供排查步骤。
```

### 4. 数据库连接失败

```
无法连接到数据库：

错误：
[粘贴错误]

DATABASE_URL 格式：
[粘贴（隐藏密码）]

数据库提供商：[Supabase/本地PostgreSQL/SQLite]

请告诉我可能的原因和解决方法。
```

### 5. 构建失败

```
运行 npm run build 失败：

错误输出：
[粘贴错误]

本地开发：[正常/不正常]

请帮我找出构建问题。
```
