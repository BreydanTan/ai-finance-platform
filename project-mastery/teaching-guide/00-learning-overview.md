# 学习路线图 - AI 财务平台完全掌握指南

**目标读者**: 非专业程序员、初学者、希望通过 AI 协作学习编程的学习者

**学习方式**: 通过 AI 助手（Claude、ChatGPT 等）协作完成

**最终目标**: 完全理解、维护、扩展本项目

2. 克隆项目到本地
3. 安装依赖
4. 启动开发服务器
5. 修改一个简单文本

## 学习方式说明

### 📚 如何使用 AI 助手学习

**有效提示词的三要素**:
1. **明确目标** - "我想实现 XXX 功能"
2. **提供上下文** - "基于 project-mastery/analysis/..."
3. **具体问题** - "请帮我解释第 123 行代码的作用"

**示例对比**:

❌ **不好的提示词**:
```
帮我修复这个错误
```

✅ **好的提示词**:
```
我在运行 npm run dev 时遇到错误：
"Error: Cannot find module 'prisma'"

我的操作系统是 Windows 11，Node.js 版本是 22.0.0。
我已经执行了 npm install。

请帮我诊断问题并给出解决步骤。
```

## 常见问题预案

### Q1: 我完全没有编程基础，能学会吗？

**A**: 可以！本指南专为非专业人员设计。建议：
1. 先花 1-2 天学习 HTML/CSS 基础（菜鸟教程）
2. 了解 JavaScript 基本语法（不需要精通）
3. 然后开始本课程，遇到不懂的概念随时问 AI

**推荐预备课程**:
- [菜鸟教程 - HTML](https://www.runoob.com/html/html-tutorial.html)
- [菜鸟教程 - JavaScript](https://www.runoob.com/js/js-tutorial.html)
- [B站 - JavaScript 入门](https://www.bilibili.com/video/BV1Sy4y1C7ha/)

---

### Q2: 遇到错误怎么办？

**步骤**:
1. **仔细阅读错误信息**（通常会告诉你问题在哪）
2. **复制错误信息 Google 搜索**
3. **查看 project-mastery/teaching-guide/resources/error-encyclopedia.md**
4. **向 AI 提问**，提供完整错误信息和上下文

**向 AI 提问模板**:
```
我遇到了以下错误：
[粘贴完整错误信息]

我在执行：[你的操作]
我的环境：[操作系统、Node版本]
我已尝试：[你的尝试]

请帮我分析原因并给出解决方案。
```

---

### Q3: 学习进度慢怎么办？

**A**: 完全正常！每个人学习节奏不同。建议：
- 不要跳过步骤
- 确保每个阶段都真正理解再继续
- 休息和复习同样重要
- 加入学习社区（Discord、微信群）寻求帮助

---

### Q4: 如何知道自己真正掌握了？

**自我评估清单** (每阶段结束后):
